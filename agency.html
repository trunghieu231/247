The Agency Portal

Sales solutions for your agents, increase revenue and achieve business goals. Simple, easy to use, fully managed by agents

The Agency Portal

Agent Registration

  • Anyone can register but the airline has to approve

Agent Options

  • Once logged in Agent has many options

Booking Flow

  • Agent details are pre-filled
  • All payment options are displayed to the agent.
  • This is defined by the airline administrator

The Agency Options

Manage agent users

  • Agent with their admin login can create multiple users with different access levels
  • They have full control to enable and disable their own users

Sub agent management

  • If a master agent, they can create multiple sub agents under them
  • As users they can fully manage their access rights and disable at any point

Agents hold bookings

  • Agents are allowed to hold a booking based on cancellation/ticketing deadline time limits
  • If not confirmed system auto cancels them
  • This helps agents to get back to their customers and confirm bookings

Agents hold bookings

  • Agents can see their full top-up payment logs and manage
  • Able to see full transaction reports and running balance

Fully managed by agents

  • All options are provided and automated
  • Agents will have full control
  • Fully transparent

Sub agent credit management

  • Master agent can share his main credit with airline with their sub agents
  • This will enable them to manage credit facilities efficiently

Booking management

  • Agents can see all their bookings and see full breakdown
  • Full booking details
  • See products and pax details

The Agency Admin

Admin

  • All tools provided to manage a B2B business efficiently

B2B Markup Management

  • Manage agent markup by types to give your prefered agents better commercials

Manage agents

  • Approve agents who have registered
  • Create new users and assign them to different types

Agent types

  • Airline create multiple agent types and assign different commercials for each of them
  • Country or market specific or type of agencies
  • This can apply for corporate clients too

Agent reports

  • Various reports available for monitoring productions, finance etc
